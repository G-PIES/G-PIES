ClusterDynamicsState ClusterDynamics::run ( gp_float  time_delta,

Runs the simulation and returns the end simulation state as a ClusterDynamicsState object.

Parameters
time_deltaThe time step for the simulation in seconds.
total_timeThe length of time that should be simulated in seconds.

run() can be called multiple times, and the simulation will resume from where it stopped after the last time run() was called.
